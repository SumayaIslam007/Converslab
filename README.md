# Plugin: Converslab Notes

Desc: “Small content items created by admin, shown on frontend in different ways”

# What it does (small but real):

1.Admin React screen to manage “Notes” (create/edit/delete/list)
2.Notes stored as Custom Post Type (converslab_note) + meta (e.g., priority, source_url)
3.REST API CRUD for notes
4.Dynamic Gutenberg block: “Converslab Notes” → shows latest notes with filters (priority, count)
